## 🎯 Icon Usage

### SVG Icons (Inline)
- **Size**: w-5 h-5 (20px) for small, w-12 h-12 (48px) for large
- **Style**: Outline (stroke-based)
- **Color**: currentColor (inherits from parent)

### Icon Placement
```
[Icon] Input Field        ← Input with left icon
Button [Icon]             ← Button with right icon
[Icon] Title              ← Section header
```
